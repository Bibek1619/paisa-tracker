## 🚀 How to Start (3 Steps)

### Step 1: Navigate to PaisaTracker
```bash
cd PaisaTracker
```

### Step 2: Install Dependencies
```bash
npm install
```

### Step 3: Start the App
```bash
npm start
```

**That's it!** Scan the QR code with Expo Go app on your phone! 📱
