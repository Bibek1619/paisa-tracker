# Backend API (Coming Soon)

This folder is reserved for the backend API when needed.

## Current Setup

Currently using **Firebase** for backend:
- Firebase Authentication
- Firestore Database
- Firebase Storage

## Future Plans

If a custom backend is needed, it will be built here with:
- Node.js + Express
- TypeScript
- MongoDB or PostgreSQL
- RESTful API
- JWT Authentication

## Why Firebase First?

Firebase provides:
- ✅ Quick setup
- ✅ Real-time database
- ✅ Built-in authentication
- ✅ Scalable hosting
- ✅ No server management

When the app grows and needs custom backend logic, we can build it here.
